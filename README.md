# Country Weather Finder

A simple React app that lets you search for countries and view their basic information along with current weather data for their capitals.

## What it does

- Search for countries by name
- Display country information (capital, area, population, languages, flag)
- Show current weather conditions for the country's capital city
- Clean, minimal user interface

## Built with

- React
- Vite
- CSS3
- Axios

## APIs used

- [REST Countries API](https://studies.cs.helsinki.fi/restcountries/) - for country data
- [Open-Meteo API](https://open-meteo.com/) - for weather data

## Getting started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and visit the local development URL

## Usage

Simply type a country name in the search box to find information about it. If multiple countries match your search, you'll see a list with "Show" buttons to view details for each one.
